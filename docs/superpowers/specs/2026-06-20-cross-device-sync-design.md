# Tablo English Lab — 기기 간 동기화 설계

- 날짜: 2026-06-20
- 대상 파일: `index.html` (단일 파일 앱)
- 목표: 혼자 여러 기기에서 사용 — "컴퓨터에서 공부하다 핸드폰으로 이어서"

## 배경 / 문제

현재 앱은 모든 데이터를 **IndexedDB**(`tabloEnglishLab`)에 저장한다. 저장소는
브라우저·기기마다 독립적이라, 컴퓨터에서 추가한 단어가 핸드폰에는 보이지 않는다.
같은 기기라도 브라우저가 다르면 공유되지 않는다.

기존에 수동 내보내기/가져오기(JSON)는 있으나([index.html:661](../../../index.html)),
매번 손으로 해야 하고 최신 데이터를 덮어쓸 위험이 있다.

## 목표

- 단어·영상·예문(**텍스트 데이터만**)이 여러 기기에서 자동으로 동일하게 보인다.
- 한쪽에서 추가/수정/삭제하면 다른 기기에도 반영된다.
- 단일 `index.html` 구조 유지(새 빌드 도구·프레임워크 없음).

## 비목표 (YAGNI)

- 녹음 오디오 동기화 — 제외. 녹음은 그 기기에만 남는다.
- 여러 사용자/공유/권한 — 제외. 단일 사용자 전제.
- 실시간(라이브) 푸시 업데이트 — 제외(향후 확장 여지). 우선은 "켤 때 + 변경 시" 동기화로 충분.
- Gemini API 키 동기화 — 제외. 키는 기기마다 따로 입력(보안상 더 안전).

## 기기 연결 방식: 동기화 코드

- 첫 기기에서 랜덤 동기화 코드 생성(예: `tablo-7h3k-9m2x`). 설정에 저장.
- 다른 기기에는 설정에서 **같은 코드**를 직접 입력.
- 같은 코드 = 같은 데이터. 로그인 화면 없음.
- 코드가 곧 접근 열쇠다. 길고 랜덤이라 추측 불가하며, 개인 학습 데이터라 위험은 낮다.
- 코드는 각 기기 IndexedDB `settings`에 보관(예: `key: 'syncCode'`).

## 무엇이 동기화되나

| 데이터 | 동기화 | 비고 |
|--------|--------|------|
| `videos` | ✅ | 영상 목록 |
| `vocab` | ✅ | 단어/예문 노트 |
| `recordings` | ❌ | 그 기기에만 |
| `settings`(Gemini 키 포함) | ❌ | 기기마다 따로 |

## Supabase 데이터 구조

테이블 2개. 각 행에 동기화용 컬럼 추가:

- `sync_code` (text) — 내 동기화 코드. 내 데이터만 필터하는 키.
- `updated_at` (timestamptz / epoch ms) — 마지막 수정 시각. 충돌 해결용.
- `deleted` (bool) — 소프트 삭제 표시. 삭제도 다른 기기에 전파.

원본 항목 필드(`id`, `videoId` 등)는 기존 IndexedDB 스키마를 그대로 옮긴다.
프로젝트 생성 시 복붙용 SQL을 제공한다.

## 동기화 동작

- **앱 시작 시(pull)**: 내 `sync_code`의 행을 받아와 IndexedDB에 병합.
- **로컬 변경 시(push)**: 단어/영상 추가·수정·삭제 시 해당 항목을 Supabase에 upsert.
  - 삭제는 행을 지우지 않고 `deleted = true` + `updated_at` 갱신.
- **충돌 처리**: 항목(행) 단위 last-write-wins. `updated_at`이 더 최신인 쪽이 이긴다.
  항목 단위라 서로 다른 단어는 영향받지 않는다.
- **오프라인**: IndexedDB가 로컬 진실원본. 오프라인에서도 앱은 정상 동작하고,
  변경은 "보내야 함(dirty)"으로 표시했다가 온라인 복귀 시 push.
- **구현**: Supabase REST(PostgREST)에 기존 코드 스타일대로 `fetch` 직접 호출.
  새 SDK/라이브러리 추가 없음.

## 사용자 최초 설정 (1회)

1. Supabase 무료 계정/프로젝트 생성.
2. 제공된 SQL로 테이블 2개 생성.
3. 프로젝트 URL + anon 키를 앱 설정에 입력(Gemini 키와 동일한 방식).
4. 동기화 코드 생성 → 다른 기기에 같은 코드 입력.

각 단계는 단계별로 안내한다.
